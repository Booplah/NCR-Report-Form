    About the Project

    This web app streamlines CrossFire Niagara’s Non-Conformance Reporting (NCR) process. Instead of manual Excel entries, users can create, view, and collaborate on NCRs with attachments (images, links) and role-based views.

    Goals: reduce time, improve consistency, and keep the whole team on the same page.
