    How to get started

    1. - Click Create NCR to start a new report.
    2. - Fill out the form, attach photo evidence (optional), and save.
    3. - Use View NCR to view existing reports.
    4. - Use the role tabs in the top bar to see role-specific views.

    Tips

    • Don't forget to fill in mandatory fields to avoid validation errors.
